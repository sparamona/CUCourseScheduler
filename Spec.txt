    - **Hide conflicts** toggle: removes any lecture section that overlaps a locked time block
    - Conflict detection is lecture-only; discussions/labs are shown in the modal

    ## Course Detail Modal

    Opened by clicking any table row.

    - Header: course ID, long title, credits, career, college
    - Distribution requirement badges (full descriptions)
    - Description
    - Prerequisites / Corequisites
    - Crosslistings
    - Grading basis
    - When offered / Last terms offered
    - Enrollment priority notes
    - **Sections table:** all sections (LEC, DIS, LAB) with days, time, instructor, open status
    - Button: **Add to My Courses**

    ## Conflict Logic

    - A lecture section is **hidden** (when "hide conflicts" is on) if its meeting time overlaps any locked course's meeting time
    - Overlap check: two time ranges overlap if `start1 < end2 && start2 < end1`, on the same day(s)
    - Discussion/lab times are not used for conflict detection (flexible enrollment)
